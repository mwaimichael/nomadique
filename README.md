Code makes up the company's dedicated and customized ERP to help with running of the business. 
Source code for our travel agency company, based and operating in Kenya & other selected destinations in Africa. 
This software is meant to solve 3 critical problems faced when it comes to business softwares like POSs. They include:
1. Inconsistent stock taking and recording
2. Hard to make updates that come with the business volatility
3. Bugs that affect effective retrieval of data from the dbs.
4. Configurations for patchworks for updates
